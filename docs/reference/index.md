# API Reference

Welcome to the ECGEN API reference documentation. This section provides detailed information about all modules, classes, and functions in ECGEN.

## Overview

ECGEN is organized into several main packages:

- **ecgen.models** - Model architectures (Generator, Discriminator, etc.)
- **ecgen.data** - Data loading and preprocessing
- **ecgen.training** - Training utilities and callbacks
- **ecgen.utils** - Helper functions and utilities
